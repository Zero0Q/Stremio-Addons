Installation

Click the "Install Addon" button above or manually add the addon URL in Stremio:

https://stremio-next-episode-addon.github.io/manifest.json

Manual Installation

If you prefer to run the addon locally:

git clone https://github.com/yourusername/stremio-next-episode-addon.git
cd stremio-next-episode-addon
npm install
npm run configure
npm start

Docker Installation

docker pull ghcr.io/yourusername/stremio-next-episode-addon:latest
docker run -p 7000:7000 -v ./config:/app/config ghcr.io/yourusername/stremio-next-episode-addon:latest
